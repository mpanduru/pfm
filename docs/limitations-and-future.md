# Limitations and Future Work

## Current Limitations

- Single-currency (RON only)
- No automatic bank syncing
- No graphical charts
- No encrypted database
- No multi-user support

---

## Possible Extensions

- CSV export
- Encrypted SQLite
- Multiple currencies
- Rule testing UI
- Budget notifications
- Charts (ASCII or graphical)

---

## Non-goals

- Cloud sync
- Mobile UI
- Web interface
