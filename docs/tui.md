# Terminal UI (TUI)

The TUI is an **interactive browser** for transactions.

It does not replace CLI commands; it complements them.

---

## Launch

```bash
pfm tui --month 2026-01 --limit 200
```

## Controls
- ↑ / ↓ or j / k — move selection
- Enter — toggle details view
- / — start typing filter
- Esc — clear filter
- q — quit

# Filtering
Filtering matches:
- Payee
- Memo
- Category
- Account
